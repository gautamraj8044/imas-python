Advanced IMASPy
===============

In this IMASPy training, we dive into more advanced features of IMASPy. It is assumed
you are familiar with the basic features of IMASPy, which are introduced in the
:ref:`IMASPy 101` training.

.. note::

    Some of the sections use the additional Python packages ``pint`` or ``xarray``.
    These can be installed by following the instructions on their websites:

    - `Installing pint <https://pint.readthedocs.io/en/stable/getting/index.html>`_
    - `Installing xarray
      <https://docs.xarray.dev/en/stable/getting-started-guide/installing.html>`_

    If you use ``pip`` you can do: ``pip install pint xarray``.
